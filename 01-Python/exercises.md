# Python Exercises

1. **Dict Comprehension:** Create a dictionary mapping characters in `sentence = "hello world"` to their frequencies.
2. **Type Hints:** Write a function `format_prompt` that takes a `str` and an optional `dict` of variables, returning a `str`. Add proper type hints.
3. **Decorator:** Write a `@retry(retries=3)` decorator that catches exceptions and retries the function execution.
4. **Generator:** Write a generator function that yields chunks of a list, simulating token streaming.
5. **AsyncIO:** Write an async function that uses `asyncio.sleep` to simulate fetching data from 3 different APIs concurrently using `asyncio.gather`.
