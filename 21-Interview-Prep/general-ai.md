# General AI / ML Questions

**Q: What is the Curse of Dimensionality?**
A: As the number of features (dimensions) increases, the volume of the space increases so fast that the available data becomes sparse, making models struggle to find statistical significance and distance metrics less meaningful.

**Q: Explain Precision vs. Recall.**
A: Precision is the fraction of relevant instances among the retrieved instances (avoiding false positives). Recall is the fraction of relevant instances that were actually retrieved (avoiding false negatives).

**Q: How do you handle imbalanced datasets?**
A: Use techniques like SMOTE for oversampling the minority class, undersampling the majority class, or using class weights in the loss function to penalize misclassifications of the minority class more heavily.

**Q: What is the difference between Generative AI and Discriminative AI?**
A: Discriminative models draw a boundary between classes (predicting Y given X, like classification). Generative models learn the distribution of the data itself (learning P(X|Y) or P(X)) to generate new instances of the data.
