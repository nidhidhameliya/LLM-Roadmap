# LLMOps (LLM Operations)

## Core Concepts
- **Tracing & Observability:** Tracking the full lifecycle of an LLM request, including prompt formatting, tool calls, and final generation time (Langfuse, LangSmith).
- **Evaluation:** Measuring LLM performance objectively (e.g., using LLM-as-a-judge for groundedness or relevance).
- **Prompt Management:** Version controlling prompts separately from the codebase.
- **Cost & Latency Tracking:** Monitoring token usage to prevent billing spikes.
- **Guardrails:** Pre/Post-processing checks to prevent toxic, off-topic, or PII data generation.
