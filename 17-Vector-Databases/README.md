# Vector Databases

## Core Concepts
- **Embeddings:** The numerical representation of text.
- **ANN (Approximate Nearest Neighbors):** Algorithms (like HNSW) that allow searching millions of vectors in milliseconds by trading a tiny bit of accuracy for massive speed gains.
- **Distance Metrics:** Cosine Similarity, L2 (Euclidean), Inner Product.
- **Metadata Filtering:** Filtering search results before or after vector search (e.g., "search only documents where author = 'John'").
