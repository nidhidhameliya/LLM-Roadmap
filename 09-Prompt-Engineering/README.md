# Prompt Engineering

## Core Techniques
- **Zero-Shot:** Direct instruction without examples.
- **Few-Shot:** Providing 1-3 examples to format the output or demonstrate the logic.
- **Chain of Thought (CoT):** "Let's think step by step." Forces the model to generate reasoning tokens before the final answer, improving accuracy on complex tasks.
- **ReAct (Reason + Act):** Combining CoT with tool usage (Thought -> Action -> Observation).
- **Prompt Chaining:** Breaking a complex task into multiple sequential prompts instead of one massive prompt.
