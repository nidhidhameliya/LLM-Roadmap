# AI Agents

## Core Concepts
- **LLM as the Brain:** The language model acts as the central reasoning engine.
- **Tool Calling (Function Calling):** The LLM is provided with schemas of external functions (e.g., search_web, query_db) and outputs JSON to execute them.
- **Memory:** Short-term (context window) and long-term (vector DBs) storage of past interactions.
- **Planning:** Breaking down complex tasks into smaller sub-tasks (e.g., using ReAct).
