# Multi-Agent Systems

## Core Concepts
- **Specialization:** Dividing a complex task among multiple agents, each with a specific persona, prompt, and set of tools (e.g., Researcher, Writer, Editor).
- **Orchestration:** Managing how agents communicate. 
    - *Hierarchical:* A manager agent delegates to worker agents.
    - *Sequential:* Agent A's output is passed directly to Agent B.
    - *Networked:* Agents converse in a shared group chat.
