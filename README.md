A concise, practical roadmap and production-grade portfolio demonstrating mastery of Large Language Models, AI Agents, and AI Engineering.

## 🏆 Capstone Projects
Beyond mini-projects, this repository contains production-ready capstones demonstrating modern LLM engineering:
- [Deep Research Agent](Capstones/01-Deep-Research-Agent/README.md) - LangGraph-based researcher with parallel search, web scraping, and automated evaluation.
- [Enterprise LLM Gateway](Capstones/02-Enterprise-LLM-Gateway/README.md) - FastAPI proxy with Redis semantic caching, rate limiting, and dynamic fallback routing.
- [Custom SLM LoRA + vLLM Serving](Capstones/03-Custom-LoRA-vLLM/README.md) - QLoRA fine-tuning script combined with a high-throughput vLLM deployment.

## How to use this repository
Start from `01-Python` and work your way down. Each folder contains:
- `README.md`: Core concepts explained concisely.
- `exercises.md`: Hands-on practice.
- `mini-project.md`: A small project to solidify knowledge.
- `interview-questions.md`: High-signal Q&A.
- `resources.md`: Best curated links.
- `code/`: Minimal working examples with evaluation scripts (`eval.py`).
